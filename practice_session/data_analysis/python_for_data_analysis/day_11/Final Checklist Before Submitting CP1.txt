Final Checklist Before Submitting Your Capstone Project
- Upload your project to GitHub.
  Make sure your entire project folder is properly pushed to a public or shared GitHub repository.

- Include a README.md file.
  Ensure your repository contains a well-written README.md using Markdown format.

- Provide a clear project overview in the README.
  Describe the purpose of your project, the main features, and how to run or use it.

- Test your application thoroughly.
  Double-check that your program runs without errors and all CRUD functionalities work as intended.

- Verify all requirements are met.
  Review the project guidelines and make sure you have fulfilled all the required components and deliverables.
